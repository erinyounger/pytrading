# 快速开始

欢迎使用xTrading量化交易系统！本指南将帮助您在5分钟内完成首次使用。

## 系统要求

### 浏览器支持
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

### 网络要求
- 稳定的互联网连接
- 支持HTTPS的浏览器

## 步骤1：注册账户

1. 访问系统首页：https://app.xtrading.com
2. 点击"注册"按钮
3. 填写注册信息：
   - 邮箱地址
   - 设置密码
   - 确认密码
4. 点击"创建账户"

## 步骤2：邮箱验证

1. 查收注册邮箱中的验证邮件
2. 点击邮件中的验证链接
3. 验证成功后自动登录系统

## 步骤3：完善用户信息

1. 登录后，系统会引导您完善个人信息
2. 填写以下信息：
   - 用户名
   - 真实姓名
   - 手机号码
   - 身份认证（可选）

## 步骤4：创建第一个策略

### 4.1 进入策略管理

1. 点击左侧导航菜单中的"策略管理"
2. 进入策略列表页面
3. 点击"新建策略"按钮

### 4.2 选择策略类型

1. 系统提供以下策略模板：
   - **MACD趋势策略**：基于MACD指标的趋势跟踪策略
   - **布林带策略**：基于布林带的均值回归策略
   - **海龟策略**：经典的海龟交易突破策略
   - **自定义策略**：创建自己的策略

### 4.3 填写策略信息

以MACD策略为例：

```
策略名称：我的MACD策略
策略描述：基于MACD指标的短期交易策略
策略类型：趋势策略
```

### 4.4 配置策略参数

```json
{
  "fast_period": 12,
  "slow_period": 26,
  "signal_period": 9,
  "atr_multiplier": 2.0
}
```

### 4.5 保存策略

1. 点击"保存"按钮
2. 策略创建成功，状态为"草稿"

## 步骤5：运行第一个回测

### 5.1 进入回测中心

1. 点击左侧导航菜单中的"回测中心"
2. 进入回测管理页面
3. 点击"新建回测"按钮

### 5.2 配置回测参数

1. **选择策略**：选择刚创建的"我的MACD策略"
2. **选择股票**：
   - 单股票模式：选择"浦发银行(SHSE.600000)"
   - 或指数模式：选择"沪深300(SHSE.000300)"
3. **设置时间范围**：
   - 开始时间：2024-01-01
   - 结束时间：2024-12-31
4. **初始资金**：1000000

### 5.3 启动回测

1. 点击"开始回测"按钮
2. 系统创建回测任务并开始执行
3. 可以在"回测任务"列表中查看进度

## 步骤6：查看回测结果

### 6.1 进入结果页面

1. 回测完成后，点击回测任务进入详情页
2. 查看回测结果摘要：
   - 总收益率：+15.23%
   - 夏普比率：1.45
   - 最大回撤：-8.5%
   - 胜率：65.2%

### 6.2 查看详细分析

1. **收益曲线**：查看累计收益率走势
2. **交易记录**：查看所有买卖交易
3. **持仓分析**：查看持仓变化
4. **风险指标**：查看各类风险指标

### 6.3 导出报告

1. 点击"导出报告"按钮
2. 选择报告格式（PDF/Excel）
3. 下载报告文件

## 步骤7：监控仪表盘

### 7.1 进入仪表盘

1. 点击左侧导航菜单中的"仪表盘"
2. 查看系统概览：
   - 总资产：¥1,152,300
   - 今日收益：+¥2,300 (+0.2%)
   - 活跃策略：3个
   - 今日交易：5笔

### 7.2 查看关键指标

1. **收益曲线**：过去30天的收益走势
2. **持仓分布**：按股票分类的资产配置
3. **最近交易**：最新的买卖记录

## 常见问题

### Q1：忘记密码怎么办？
A：点击登录页面的"忘记密码"链接，通过邮箱重置密码。

### Q2：如何更改策略参数？
A：进入策略管理页面，点击策略名称进入详情页，点击"编辑"按钮修改参数。

### Q3：回测需要多长时间？
A：取决于股票数量和时间范围，一般1-10分钟。

### Q4：可以同时运行多个回测吗？
A：可以，系统支持批量回测和并行执行。

### Q5：如何联系技术支持？
A：点击系统右下角的"帮助"图标，或发送邮件到 support@xtrading.com。
